Linguagem server-side: PHP
Banco de dados: MySQL

Instruções para funcionamento:

- Configurar no arquivo config/config.php a função BDconecta() as variáveis com os dados de acesso do Banco de Dados MySQL.
- Criar base de dados com nome 'servicenet'.
- Importar o arquivo 'servicenet.sql', disponível no GitHub, na base de dados.
